Face Book

Open the Yale Face Book. Open your browser's developer tools (F12, Command+Option+I, or Right Click + Inspect Element), and navigate to the Network tab. Then, log in, and choose a network request that occurred after the login and copy the value of the Cookie property.

Directory

Open the Yale Directory, and open your developer tools again. Log in (you may still need to click the blue LOG IN button to access the privileged version of the website). Then, perform a search, and in the developer tools, select the query to the api endpoint. You'll notice the Cookie is too long to be displayed without elipses, so right click and copy it elsewhere then extract only the _people_search_session value. (You can also copy the value under the "Cookies" tab of the developer tools.) Finally, grab the X-CSRF-Token header value.
